# Mandarin

Site completo para curso personalizado de mandarim.

## Objetivo

Curso online acessado pelo navegador, com 180 dias de estudo, áudio, repetição, revisão inteligente, gravação de voz, testes e simulações para viagem à China, feira, fornecedores, restaurante, hotel e negociação.

## Publicação recomendada

Publique gratuitamente na Vercel, Netlify ou Cloudflare Pages.

## Vercel

Framework: Vite
Build Command: npm run build
Output Directory: dist

## Comandos locais

npm install
npm run dev
npm run build
